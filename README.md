# COVID-19-Cases-and-Global-Vaccination-Analytics-Dashboard
Developed a Power BI dashboard to analyze COVID-19 cases and global vaccination progress. Created KPIs, trend analysis, and interactive visuals to monitor infections, recoveries, deaths, and vaccination coverage.

# Project Objective
To analyze COVID-19 cases, recoveries, deaths, and vaccination progress using Power BI. The dashboard helps track pandemic trends, compare state and country performance, and evaluate global vaccination coverage through interactive visualizations and KPIs.

# Questions (KPIs)
What is the total number of confirmed COVID-19 cases?
What is the total number of recoveries?
What is the total number of deaths?
How many vaccine doses have been administered globally?
What percentage of the population is fully vaccinated?
Which states/countries have the highest cases and vaccination rates?
How have COVID-19 cases changed over time?

# Process
- Imported COVID-19 time-series, Indian state-wise, and vaccination datasets into Power BI.
- Cleaned and transformed data using Power Query.
- Created a Date Table and established relationships between datasets.
- Developed DAX measures for KPIs and vaccination metrics.
- Built interactive dashboards using cards, charts, treemaps, slicers, and trend analysis visuals.
- Generated insights on COVID-19 spread and vaccination progress.

# Project Insight
- Vaccination coverage varied significantly across countries and WHO regions.
- States with high confirmed cases generally showed higher recovery counts.
- COVID-19 trends revealed multiple waves of infections over time.
- Countries that adopted vaccination earlier achieved higher vaccination coverage.
  
# Final Conclusion
The dashboard provides a comprehensive view of COVID-19 cases and vaccination progress. It enables users to monitor pandemic trends, compare regional performance, and evaluate vaccination effectiveness through interactive and data-driven visualizations.
